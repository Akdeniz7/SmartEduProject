# SmartEduProject

## Features

### ROLE
#### Admin
* Add Course
* Add Category
* Add Course
* Delete Course
* Delete User

#### Teacher
* Add Course
* Edit Course
* Delete Course
* Enroll Course
* Relase Course

#### Student
* Enroll Course
* Relase Course

### PAGES
* HOME
* LOGIN
* REGISTER
* DASHBOARD (ADMIN , TEACHER, STUDENT)
* ABOUT
* CONTACT (EMAIL SERVICE)
* COURSES (SEARCH)
* SINGLE-COURSE
